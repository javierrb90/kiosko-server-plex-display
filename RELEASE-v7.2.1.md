# BBQ v7.2.1 — Grupos por Subtipo

Esta versión simplifica la creación de grupos.

## Tipos de grupo

- **Manual**: la pertenencia se gestiona desde la ficha de cada item.
- **Dinámico**: incluye automáticamente items cuyo campo **Subtipo** contiene el valor configurado.
- **Mixto**: combina pertenencia manual y coincidencias automáticas por Subtipo.

El formulario solo muestra el valor dinámico al elegir Dinámico o Mixto. La comparación no distingue mayúsculas y minúsculas.
