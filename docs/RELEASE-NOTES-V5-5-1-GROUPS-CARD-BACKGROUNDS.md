# v5.5.1 · Agrupación temporal y tarjetas configurables

- Backlog agrupado visualmente por HOY, AYER, ÚLTIMA SEMANA y ANTERIOR.
- Los encabezados de fecha ocupan una celda de ancho completo dentro del grid.
- Añadidos controles de diseño para el fondo de cada item:
  - activar/desactivar backdrop dentro de tarjetas;
  - opacidad;
  - blur;
  - oscurecimiento.
- Eliminado el texto de fuente dentro de cada tarjeta.
- La fuente se distingue por el color de acento lateral configurable.
- Eliminada la etiqueta “Visto antes / Terminado antes”; las estrellas ya indican que existe una valoración previa.
