# v5.7.2 · Actual con estado y estrellas

- La vista Actual muestra estado del item:
  - Backlog;
  - On Deck;
  - estrellas si ya está en Colecciones.
- Sustituido el botón `Marcar como terminado/visto` por estrellas visibles.
- Al pulsar una puntuación se abre el flujo de confirmación/guardado.
- El botón `Añadir a On Deck` queda debajo de las estrellas.
- La mini-card de Actual se fuerza a ocultarse al entrar en la vista Actual.
