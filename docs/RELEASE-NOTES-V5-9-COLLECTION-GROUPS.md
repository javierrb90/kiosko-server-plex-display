# v5.9 · Grupos en Colecciones

- Añadida persistencia de grupos en `data/collection-groups.json`.
- Nuevos endpoints `/api/collection-groups`.
- Grupos manuales, dinámicos y mixtos.
- Reglas dinámicas iniciales por:
  - plataforma;
  - género;
  - desarrollador;
  - publisher;
  - año;
  - fuente;
  - tipo;
  - título.
- En filtros de Colecciones se pueden seleccionar grupos.
- Modo de coincidencia: cualquiera / todos.
- Se pueden crear grupos desde el panel de filtros.
- Desde la ficha de un item en Colecciones se puede añadir/quitar de grupos manuales/mixtos.
