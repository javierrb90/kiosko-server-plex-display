# v5.6.4 · Visual unificado y fondos

- Unificadas las pestañas Diseño/Fondos en una sola pestaña Visual.
- Mejoradas las tabs de opciones en responsive: ahora se organizan en grid/wrap y no aparece la barra horizontal rota.
- El fondo de On Deck usa el mismo sistema visual que Backlog/Colecciones.
- La vista Actual usa las mismas reglas de fondo:
  - opacidad;
  - blur;
  - oscurecimiento;
  - escala de grises;
  - fade.
- Añadidos controles para el fondo de vistas:
  - oscurecimiento;
  - blanco y negro;
  - fade más suave.
- Las transiciones entre fondos son más suaves.
