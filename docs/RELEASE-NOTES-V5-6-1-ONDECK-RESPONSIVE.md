# v5.6.1 · On Deck series + responsive grid

- On Deck canonicaliza Plex igual que Colecciones:
  - episodio/temporada/show se guardan como serie;
  - se conserva el episodio original en `meta`.
- Al mandar un episodio desde Backlog a On Deck entra la serie.
- Al mandar Actual a On Deck también se canonicaliza como serie cuando procede.
- Responsive móvil vuelve a grid vertical compacto:
  - S: 2 columnas;
  - M: 3 columnas;
  - L: 4 columnas.
- En escritorio S/M/L ahora cambian ancho, poster y altura de tarjeta.
