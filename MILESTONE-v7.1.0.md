# BBQ v7.1.0 — Milestone SQLite

Esta versión cierra la primera fase estable de BBQ sobre SQLite. Reúne persistencia estructurada, assets externos, API de ingestión, integraciones locales, backups y los flujos principales de organización.

## Cambio visible de esta entrega

El contador y botón Dar la vuelta ya no se renderizan en Actividades ni en Colección, tanto en diseño Normal como Simple. Solo aparecen en Backlog y On Deck.

## Documentación

La carpeta `docs/` ha sido reorganizada y ampliada. `docs/HANDOFF.md` contiene las decisiones, riesgos y siguientes pasos para continuar el proyecto sin el historial anterior.
